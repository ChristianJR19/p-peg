# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.5.x   | yes       |
| < 0.5   | no        |

## Reporting a Vulnerability

If you find a security issue, do NOT open a public issue.

Email: yordereal@gmail.com

Include steps to reproduce and expected vs actual behavior. You should get a response within 48 hours.
